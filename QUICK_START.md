## ğŸ”§ Troubleshooting

### Images Not Showing?

- Check that the `images/` folder is in the same directory as `index.html`
- Verify all image files are present (24 files total)
- Check browser console for errors (F12)

### Page Looks Broken?

- Make sure `styles.css` is in the same directory as `index.html`
- Clear your browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
- Try opening in a different browser

### JavaScript Not Working?

- Ensure `script.js` is in the same directory as `index.html`
- Check browser console for errors (F12)
- Make sure JavaScript is enabled in your browser
